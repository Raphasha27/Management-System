# Management System

IT infrastructure monitoring and management dashboard built with Next.js.

## Tech Stack

- **Frontend:** Next.js 16, React 19, TypeScript, Recharts
- **Backend:** Next.js API routes, Prisma ORM
- **Database:** PostgreSQL (via Prisma)
- **Auth:** JWT-based authentication

## Features

- Dashboard with real-time metrics and charts
- Client and project management
- Resource and service monitoring
- AI-powered support assistant
- Role-based access control

## Getting Started

```bash
git clone https://github.com/Raphasha27/Management-System.git
cd Management-System
npm install
cp .env.example .env.local
npx prisma migrate dev
npm run dev
```
