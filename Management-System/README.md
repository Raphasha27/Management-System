# School Management System

A simple web application to manage school data.

## Features
- **Dashboard**: View total counts of students and teachers.
- **Student Management**: Add and view student details.
- **Teacher Management**: Add and view teacher details.
- **Target Deployment**: Streamlit Community Cloud.

## How to Run Locally
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```
